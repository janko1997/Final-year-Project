What is the purpose of my Web Application?

The purpose of my Web Application is to help children with autism who struggle

with communication, to learn daily and body movement activities, answer questions using

the "Yes or No" page and express their feelings.


Usefull Tips:

•Use the "Shuffle" button to make sure the child learned the activities.

•Use the "Hide Text" button to see if the child can identify the activity by reading

only the text and accordingly the "Hide Images" button.
